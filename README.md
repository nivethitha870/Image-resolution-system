Image Enhancer Pro
    An interactive AI-powered image enhancement web application built using Streamlit, OpenCV, and NumPy. This tool allows users to upload images and apply various enhancement techniques such as sharpening, denoising, contrast adjustment, and edge detection with real-time visualization.

Features
📤 Upload images (JPG, PNG, JPEG)
🎨 Image enhancement techniques:
    Brightness & Contrast Adjustment
    Sharpening
    Denoising
    Edge Detection
📊 Visualization using Plotly
⚡ Real-time processing
🖥️ User-friendly interface with sidebar controls

🛠️ Technologies Used
Python
Streamlit
OpenCV (cv2)
NumPy
PIL (Python Imaging Library)
Plotly
